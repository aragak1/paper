欢迎来到后台管理系统！


使用须知：

请小心进行操作，尤其是删除数据，一切操作都是不可恢复的！


如何修改公司的图片：

在图片管理中上传公司的图片，然后在公司中修改图片路径即可，只需要修改/static/img/company/company.png中最后的company.png部分。


如何添加管理员：

不要在后台直接添加，在首页主页一个新的用户，然后把用户的admin字段改为1.


如何添加试卷：

请注意试卷的名称不能重复。

导入试卷的格式 （json）：

{
    "company": "公司名称",
    "name": "试卷名称",
    "content": [
        {
            "answer": "答案（A/B/C/D）",
            "type": "单选",
            "name": "问题的题干",
            "options": [
                {"option_name": "A", "content": "选项A的内容"},
                {"option_name": "B", "content": "选项B的内容"},
                {"option_name": "C", "content": "选项C的内容"},
                {"option_name": "C", "content": "选项D的内容"} ……
            ]
        }，
